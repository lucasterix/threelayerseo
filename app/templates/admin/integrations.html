Google Search Console

Ermöglicht (a) automatisches Einreichen der sitemap.xml und (b) DNS-basierte Domain-Verification beim Erstellen einer neuen Site. Einmaliges Setup:

  1. In der Google Cloud Console ein Projekt erstellen, die APIs Search Console API und Site Verification API aktivieren.
  2. OAuth-Client anlegen (Web application), Redirect-URI https://seo.zdkg.de/integrations/gsc/callback eintragen.
  3. Client-ID + Secret in die Server-.env: GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET. Admin-Container neu starten.
  4. Unten auf „Connect" klicken, Google-Consent abnicken, den angezeigten Refresh-Token in die .env als GOOGLE_REFRESH_TOKEN eintragen und Admin neu starten.
