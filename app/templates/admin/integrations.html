Google Search Console (Service Account)

Auto-Onboarding nach Domain-Kauf: DNS-TXT via INWX → Google-Verification → Site-Property anlegen → sitemap.xml einreichen. Läuft ohne OAuth-Consent und ohne Ablaufzeit, solange der Service-Account-Key gültig ist.

  1. GCP-Projekt erstellen (z.B. „threelayerseo").
  2. APIs aktivieren: Search Console API und Site Verification API.
  3. Service-Account anlegen (Name: „threelayerseo-gsc", keine Rolle nötig).
  4. Auf dem SA → Keys → Add Key → JSON. Das JSON landet im Download-Ordner.
  5. Lokal: scp ~/Downloads/threelayerseo-*.json threelayerseo:~/apps/threelayerseo/secrets/google-sa.json
  6. Server: docker compose up -d admin renderer worker
